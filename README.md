# Pandas ilə Satış Analizi Tapşırığı

Bu tapşırıqda sizin məqsədiniz təqdim olunan satış məlumatlarını Pandas kitabxanası vasitəsilə analiz etməkdir.

### Tapşırıqlar:
1. `sales_data.csv` faylını Pandas-da DataFrame olaraq oxuyun.
2. Məlumatları **"Kateqoriya"** sütununa görə qruplaşdırın.
3. Hər kateqoriya üzrə **"Satış Məbləği"** sütununun cəmini hesablayın.
4. Alınan nəticəni `kategorya_uzre_satis.csv` adı ilə yadda saxlayın.

### Tələblər:
- Kodunuzu `solution.py` və ya `analiz.ipynb` faylında yazın.
- İşi bitirdikdən sonra GitHub-a "Commit" və "Push" etməyi unutmayın.
